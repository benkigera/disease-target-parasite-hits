# Disease Target Parasite Hits

Technical notes are hosted as a GitHub Pages HTML page:

https://benkigera.github.io/disease-target-parasite-hits/

The atlas is generated by two pipeline stages:

```bash
bun run pipeline:targets
bun run pipeline:viruses
```

The second stage writes the same generated dataset to both the pipeline output folder and the deployed app's `data` folder. It currently retains only Tier A direct viral interactions supported by structural or quantitative assays.

Raw HPIDB MITAB input files are not tracked because they exceed GitHub's file-size limit. Place them locally under:

```text
disease-target-virus-pipeline/data/inputs/hpidb2.mitab/
```
